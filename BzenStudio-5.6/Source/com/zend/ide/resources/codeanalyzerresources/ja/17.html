カテゴリ： セキュリティ
グローバル変数は、その変数自身が定義される前に使用されることがあります。 register_globals　変数の設定あるいは、外部のインクルードファイル内に変数が 定義されている場合にその事象に関する警告メッセージが出力されるかもしれません。 また、それらの警告メッセージを無視するように設定することもできます。 しかしながら、このような振る舞いは、ある環境の中でコードを実行した時、予測が つかない振る舞いをする場合もあることに注意してください。

サンプル

// $a was never initialized!
if ($a > 0) {
    ...
}
