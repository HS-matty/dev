カテゴリ： バグ
条件分岐構文内での値の割り当ては、時々、演算結果のデータ型が割り当てられます。 多くの場合は、割り当て演算子 = の代わりに、等式演算子 == が使用されます。 変数と変数以外の表記を比較する場合、等式演算子の右辺の値を変数に格納することが共通の振る舞いです。その結果、等式演算子が誤って割り当て演算子と取り違えられると PHP は、即座にパースエラーを報告します。これは、このクラスの大部分のバグを回避するのに役立ちます。

サンプル

if ($a = 1) { // probably it was meant to be $a == 1 here
    ...
}
