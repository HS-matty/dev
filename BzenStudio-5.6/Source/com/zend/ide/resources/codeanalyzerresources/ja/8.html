説明

カテゴリ： バグ
if 文ブロックを示す大括弧({})をなしで if 文の後に二つの入れ子で else 文が使用されている場合、ひとつの else 文は構文上、誤りがあるかもしれません。これからを回避するには、いつも if 文ブロックを示す大括弧({})を使用することをお勧めします。

サンプル

Error prone if ($a == 0)
    if($c > 0)
        print "OK";
    else
    print "a is not zero!";
Recommended if ($a == 0) {
    if($c > 0) {
        print "OK";
    } else {
        print "a is not zero!";
    }
}
