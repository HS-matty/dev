Description

Catégorie : Bug
Lorsque else est utilisé après deux structures if imbriquées sans délimiteurs de blocs (accolades), il est possible de se tromper d'association if-else. Il est toujours préférable d'utiliser les accolades.

Exemple

Sujet à erreur if ($a == 0)
    if($c > 0)
        print "OK";
    else
    print "a n'est zéro!";
Recommandé if ($a == 0) {
    if($c > 0) {
        print "OK";
    } else {
        print "a n'est pas zéro!";
    }
}
