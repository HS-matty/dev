Description

Catégorie : Sécurité
Une variable globale est utilisée avant qu'elle ne soit définie. Vous pouvez rencontrer cette alerte si vous utiliser l'option de register_globals ou des variables définies dans des fichiers externes inclus, auquel cas, vous pouvez l'ignorer. Néanmois, notez que cela peut conduire votre code à des comportements imprédictibles dans certains contextes.

Exemple

// $a n'est jamais initialisé!
if ($a > 0) {
    ...
}
