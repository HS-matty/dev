Description

Catégorie : Performance
Une variable est passée par référence, mais sa valeur n'est pas modifiée. Contrairement à la croyance populaire, cela réduit les performances dans la plupart des cas, et il est préférable d'utiliser la méthode de passage par valeur standard.

Exemple

Inefficace function foo(&$a)
{
    echo $a;
}
Recommandé function foo($a)
{
    echo $a;
}
