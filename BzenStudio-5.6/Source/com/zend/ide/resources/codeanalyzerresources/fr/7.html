Catégorie : Fonctionnalité obsolète
Le passage par référence à la volée est obsolète dans les versions les plus récentes de PHP. Au lieu de spécifier le préfixe & dans l'appel de la fonction, ajoutez le dans le prototype de la fonction.

Exemple

Obsolète function increment($n)
{
    $n++;
}

$foo = 5;
increment(&$foo);
Correct function increment(&$n)
{
    $n++;
}

$foo = 5;
increment($foo);
