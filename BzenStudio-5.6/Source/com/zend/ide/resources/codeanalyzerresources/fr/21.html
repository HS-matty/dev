Description

Catégorie : Bug
Lorsqu'une fonction retourne parfois une valeur, et parfois pas (i.e. utilise une instruction return; vide), cela peut impliquer que la fonction ne se comporte pas comme il faut, dans certaines circonstances. Alternativement, cela peut signifier que la valeur retournée est redondante.

Exemple

if ($a) {
    return 1;
} else {
    return;
}
