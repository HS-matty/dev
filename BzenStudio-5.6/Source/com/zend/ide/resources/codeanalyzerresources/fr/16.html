Catégorie : Style de codage
Une variable locale est utilisée avant d'être définie. Même si PHP initialise les variables avec des valeurs vides lors de la première utilisation, il est considéré comme une pratique courante d'initialiser les variables, pour rendre le code plus lisible, et éviter des bugs.

Exemple

function foo()
{
    // $a n'est jamais initialisé
    if ($a > 0) {
        ...
    }
}
