Description

Catégorie : Bug
Une fonction est définie comme retournant une valeur par référence, mais le résulat n'est pas assigné par référence. Cela signifie que la référence est cassée, et que la valeur retournée sera dupliquée lors de l'affectation.

Exemple

Incorrect function &foo()
{
    $ret = new X();
    return $ret;
}

$a = foo();
Correct function &foo()
{
    $ret = new X();
    return $ret;
}

$a =& foo();
