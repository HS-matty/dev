Description

Catégorie : Bug
L'argument passé à la structure break est plus grand que le niveau d'imbrication actuel.

Exemple

function example()
{
    for ($i=0; $i<100; $i++) {
        for ($j=0; $j<100; $j++) {
            if (($i * $j) > 1000) {
                break 3;        // Le niveau d'imbrication à ce niveau ne peut être que 2
            }
        }
    }
}
