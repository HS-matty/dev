Description

Catégorie : Bug
La structure break permet aux développeurs d'interrompre l'imbrication de plusieurs boucles en spécifiant un argument optionnel. En passant une variable à break peut modifier la façon avec laquelle votre code s'exécute de manière très inattendue, et doit être évité.

Exemple

Dangeureux break $foo;
Alternative if ($foo==1) {
    break;
} else if ($foo=2) {
    break 2;
} else {
    die("Erreur inattendue");
}
