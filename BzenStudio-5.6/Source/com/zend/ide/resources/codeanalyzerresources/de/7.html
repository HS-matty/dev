Kategorie: Deprecated Functionality
Call-Time Pass-by-Reference ist in neueren PHP-Versionen "deprecated". Anstatt den Parameter beim Funktionsaufruf mit einem &-Präfix zu versehen, geben Sie dies bei der Funktionsdeklaration an.

Beispiel

Deprecated function increment($n)
{
    $n++;
}

$foo = 5;
increment(&$foo);
Korrekt function increment(&$n)
{
    $n++;
}

$foo = 5;
increment($foo);
