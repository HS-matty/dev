Beschreibung

Kategorie: Security
Eine globale Variable wird verwendet, bevor sie initialisiert wurde. Diese Warnung erscheint möglicherweise, weil Sie sich auf register_globals oder Variablen aus Include-Dateien verlassen. In diesen Fällen können Sie diese Warnung ignorieren.

Beispiel

// $a was never initialized!
if ($a > 0) {
    ...
}
