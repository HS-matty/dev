Kategorie: Bug
break erlaubt es, mit einem optionalen Parameter aus verschachtelten Schleifen auszubrechen. F? diesen Parameter sollte keine Variable verwendet werden, da dies zu unerwarteten Seiteneffekten f?ren kann.

Beispiel

Gefährlich break $foo;
Alternative if ($foo==1) {
    break;
} else if ($foo=2) {
    break 2;
} else {
    die("Unexpected error");
}
