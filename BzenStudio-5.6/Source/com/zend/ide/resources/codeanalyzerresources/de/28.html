Beschreibung

Kategorie: Security
include() oder eine analoge Funktion wird mit einem variablen Parameter verwendet. Dies kann gefährlich sein, da Variablen in vielen Fällen von aussen beeinflusst werden oder versehentlich ?erschrieben werden können. Es wird geraten, Konstanten anstelle von Variablen zu verwenden.

Beispiel

Gefährlich $script_path = "/htdocs";
include($script_path."/foo.inc");
Empfohlen define('SCRIPT_PATH', "/htdocs");
include(SCRIPT_PATH."/foo.inc");
