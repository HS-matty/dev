Beschreibung

Kategorie: Bug
Eine Funktion wurde als "returning-by-reference" definiert, das Ergebnis wird aber nicht "by-reference" zugewiesen. Dies f?rt dazu, dass eine Kopie anstelle einer Referenz zur?kgegeben wird.

Beispiel

Falsch function &foo()
{
    $ret = new X();
    return $ret;
}

$a = foo();
Richtig function &foo()
{
    $ret = new X();
    return $ret;
}

$a =& foo();
