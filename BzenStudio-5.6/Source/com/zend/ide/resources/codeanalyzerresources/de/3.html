Beschreibung

Kategorie: Bug
Zuweisungen in einem Konditionalausdruck sind manchmal ein Tippfehler. In vielen Fällen soll eigentlich der Gleichheitsoperator == anstelle des Zuweisungsoperators = benutzt werden. Um diesem Fehler vorzubeugen können Sie die Variable auf die rechte Seite des Ausdrucks setzen. In diesem Fall meldet PHP dann einen Parser-Fehler, falls der Zuweisungsoperator versehentlich verwendet wurde.

Beispiel

if ($a = 1) { // Sollte möglicherweise $a == 1 heissen.
    ...
}
