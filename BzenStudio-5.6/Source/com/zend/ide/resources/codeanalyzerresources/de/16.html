Kategorie: Coding Style
Eine lokale Variable wird vor ihrer Deklaration verwendet. Obwohl PHP Variablen bei ihrer ersten Verwendung initialisiert, gilt es als guter Stil, Variablen explizit zu initialisieren. Dies verbessert die Lesbarkeit des Codes und beugt Fehlern vor.

Beispiel

function foo()
{
    // $a wurde nicht initialisiert!
    if ($a > 0) {
        ...
    }
}
