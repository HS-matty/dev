Beschreibung

Kategorie: Bug
Wenn else nach zwei geschachtelten ifs ohne Blöcke (geschweifte Klammern) verwendet wird, kann das else-Statement mit der if-Präposition verwechselt werden. Daher ist es besser, immer geschweifte Klammern zu verwenden.

Beispiel

F? Fehler anfällig if ($a == 0)
    if($c > 0)
        print "OK";
    else
    print "a is not zero!";
Emphohlen if ($a == 0) {
    if($c > 0) {
        print "OK";
    } else {
        print "a is not zero!";
    }
}
