Descripción

Categorú}: Bug
Cuando else es usado después de dos ifs aninados sin delimitadores de bloque (paréntesis de llave), uno podrú} confundir a que preposición "if" corresponde el "else". Es mejor especificar siempre las llaves.

Ejemplo

Propenso a errores if ($a == 0)
    if($c > 0)
        print "OK";
    else
    print "a no es cero!";
Recomendado if ($a == 0) {
    if($c > 0) {
        print "OK";
    } else {
        print "a no es cero!";
    }
}
