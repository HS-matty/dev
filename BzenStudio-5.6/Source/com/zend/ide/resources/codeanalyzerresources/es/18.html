Categorú}: Bug
Un de las variables $HTTP_*_VARS, como $HTTP_GET_VARS, est?siendo usada. A diferencia de las nuevas variables de entrada ($_GET, $_POST, etc.), las antiguas variables $HTTP_*_VARS not están explú€itamente disponibles in las funciones. Para poder acceder a ellas, debe declararlas como globales, usar el array $GLOBALS, o usar las nuevas $_GET/$_POST/$_COOKIE/$_SERVER/$_ENV/$_SESSION (recomendado).

Ejemplo

Erroneo function hola_mundo()
{
    if (empty($HTTP_GET_VARS['name'])) {
        // $HTTP_GET_VARS en este contexto est?siempre sin definir
        print "Hola, extraño!";
    } else {
        print "Hola, $HTTP_GET_VARS[name]";
    }
}
Correcto
alternativa #1 		function hola_mundo()
{
    global $HTTP_GET_VARS;
    
    if (empty($HTTP_GET_VARS['name'])) {
        print "Hola, extraño!!";
    } else {
        print "Hola, $HTTP_GET_VARS[name]";
    }
}

alternativa #2
(recomendada) 		function hola_mundo()
{
    if (empty($_GET['name'])) {

    } else {
        print "Hola, $_GET[name]";
    }
