Categorú}: Rendimiento
Una variable es pasada por referencia pero su valor no es modificado. Contrariamente a la creencia popular, esto incurre en una reducción del rendimiento en la mayorú} de los casos, por ello es preferible usar el estándar de paso-por-valor.

Ejemplo

Ineficiente function foo(&$a)
{
    echo $a;
}
Recomendado function foo($a)
{
    echo $a;
}
