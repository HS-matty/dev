Descripción

Categorú}: Bug
El argumento especificado para break is mayor que el actual nivel de anidamiento.

Ejemplo

function ejemplo()
{
    for ($i=0; $i<100; $i++) {
        for ($j=0; $j<100; $j++) {
            if (($i * $j) > 1000) {
                break 3;        // El nivel de anidamiento aqu?es sólo de 2
            }
        }
    }
}
