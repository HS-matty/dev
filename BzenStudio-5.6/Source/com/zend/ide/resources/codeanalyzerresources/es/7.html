Descripción

Categorú}: Funcionalidad Obsoleta
Call-time pass-by-reference ha sido declarado obsoleto en las últimas versiones de PHP. En vez de especificar el prefijo & in la llamada a la función, uselo en la declaración de la función.

Ejemplo

Obsoleto function increment($n)
{
    $n++;
}

$foo = 5;
increment(&$foo);
Correcto function increment(&$n)
{
    $n++;
}

$foo = 5;
increment($foo);
