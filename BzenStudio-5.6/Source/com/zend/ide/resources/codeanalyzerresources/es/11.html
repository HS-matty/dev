Descripción

Categorú}: Bug
El contructor break permite a los desarrolladores saltar fuera de múltiples sentencias de bucle, especificando un argumento opcional. El pasar una variable a break como argumento puede cambiar el flujo del código de manera inesperada y debe ser evitado.

Ejemplo

Peligroso break $foo;
Alternativa if ($foo==1) {
    break;
} else if ($foo=2) {
    break 2;
} else {
    die("Error inesperado");
}
