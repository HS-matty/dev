Descripción

Categorú}: Bug
Una función es definida para que retorne-por-referencia, pero el resultado no es asignado por referencia. Esto significa que la referencia se romper? y el valor retornado estar?duplicado en vez de asignado.

Ejemplo

Incorrecto function &foo()
{
    $ret = new X();
    return $ret;
}

$a = foo();
Correcto function &foo()
{
    $ret = new X();
    return $ret;
}

$a =& foo();
